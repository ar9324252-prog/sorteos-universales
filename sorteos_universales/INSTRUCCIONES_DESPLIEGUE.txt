## 📋 GUÍA RÁPIDA DE DESPLIEGUE (3 PASOS)

### PASO 1: Prepare el Repositorio en tu PC

1. Abre una terminal en la carpeta del proyecto:
```
cd c:\Users\sebasti\Downloads\andy\sorteos_flask\sorteos_universales
```

2. Inicializa Git:
```
git init
git add .
git commit -m "Sorteos Universales - Listo para despliegue"
```

### PASO 2: Sube a GitHub

1. Ve a https://github.com/new
2. **Nombre del repositorio:** `sorteos-universales`
3. Descripción: "Plataforma de rifas y sorteos online"
4. Selecciona **Public** (para que Render pueda verlo)
5. Click en **Create repository**

6. En tu terminal, ejecuta:
```
git remote add origin https://github.com/TU_USUARIO/sorteos-universales.git
git branch -M main
git push -u origin main
```

### PASO 3: Deploy en Render (Gratuito)

1. Ve a https://render.com
2. Haz login con GitHub (click en "Continue with GitHub")
3. Da permiso a Render para acceder a tus repositorios
4. Click en **New +** en el dashboard
5. Selecciona **Web Service**
6. Elige **sorteos-universales** de tu lista de repos
7. Rellena:
   - **Name:** sorteos-universales
   - **Region:** cualquiera (ej: Ohio)
   - **Branch:** main
   - **Runtime:** Python 3
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
   - **Free plan** está bien para comenzar

8. Click en **Create Web Service**

**¡Listo!** En 2-3 minutos tendrás tu sitio en vivo en:
```
https://sorteos-universales-xxxx.onrender.com
```
